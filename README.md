## VS Code Settings

- ctrl-shift-p
- type "settings" in the search field
- select "Preferences: Open Workspace Settings (JSON)"
- type in settings:

  ```json
  {
    "editor.detectIndentation": false,
    "editor.indentSize": "tabSize",
    "[css]": {
      "editor.tabSize": 2
    },
    "[html]": {
      "editor.tabSize": 2
    },
    "[javascript]": {
      "editor.tabSize": 2
    },
    "[typescript]": {
      "editor.tabSize": 2
    },
    "[json]": {
      "editor.tabSize": 2
    },
    "[markdown]": {
      "editor.tabSize": 2
    }        
  }
  ```
